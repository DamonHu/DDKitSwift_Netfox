## Introduction

The plugin for viewing network request logs in [DDKitSwift](https://github.com/DamonHu/DDKitSwift). relies on [NetFox](https://github.com/kasketis/netfox).

## Regist

```
let tool = DDKitSwift_Netfox()
DDKitSwift.regist(plugin: tool)
```
